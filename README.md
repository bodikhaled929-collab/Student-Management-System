# 🎓 Student Information System

A clean and modular Student Information System developed using Python. This project demonstrates the fundamentals of Object-Oriented Programming (OOP), file handling, and data management using JSON.

---

## 📌 Features

- ➕ Add new students
- 📋 View all student records
- 🔍 Search students by ID
- ✏️ Update student information
- ❌ Delete student records
- 💾 Save data automatically using JSON
- 🧩 Modular project structure
- 🛡️ Input validation
- 📂 Clean and maintainable code

---

## 🛠️ Technologies Used

- Python 3
- Object-Oriented Programming (OOP)
- JSON
- File Handling
